34 /* Architectural note: leaves of the AST are represented as small structs,
35  * with a few public data members and a constructor taking corresponding
36  * arguments. Subtrees (for example, Expression) are boost::variants of those
37  * other AST classes.
38  *
39  * Any of the AST classes which itself contains a potentially large AST class
40  * should be held only as a std::unique_ptr, and AST classes that are not
41  * known to be leaves should never be copied.
42  */
